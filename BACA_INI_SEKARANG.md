## ANDA HARUS MELAKUKAN INI MANUAL:

Saya tidak bisa menginstall Android Studio secara otomatis karena filenya sangat besar (Hampir 1GB) dan butuh interaksi layar.

1. **JANGAN TUTUP VS CODE INI**.
2. **Download Android Studio** di sini:
   ðŸ‘‰ [https://developer.android.com/studio](https://developer.android.com/studio)
3. **Install** filenya.
   - Klik Next terus.
   - **PENTING**: Pastikan centang "Android SDK" dan "Android Virtual Device".
4. **Buka Android Studio** setelah install selesai.
   - Ikuti wizard di layar sampai dia selesai mendownload komponen tambahan.
